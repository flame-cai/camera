Method: Silhouette boundary detection via canvas pixel analysis extracts body keypoints (head top, shoulders, hips, knees, ankles). Absolute scale is derived from known camera height using pinhole projection: D = (H_cam × f) / y_ankle_px. Height uses head-to-ankle pixel span × scale. Weight is estimated via Lorentz formula adjusted for sex and measured widths. MUAC uses upper-arm pixel width × scale × π/2 (half-circumference model). WHO thresholds: SAM <11.5 cm, MAM 11.5–12.5 cm.
